1. 앱 개요
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 이름: DAOApp
- 목적: 포인트 기반 랭킹 + 경기 관리 시스템
- 핵심: 실시간 반영 + 달력 UX + 관리자 편의성
- 테마: 진한 파랑 (0xFF1565C0) 

2. 주요 기능
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 회원
   - 구글 로그인 (Firebase Auth)
   - admin 연동 setadmin js연동/ Custom Claims
   - 

2. 포인트 시스템
   - 관리자 수동 부여 / 수정 / 삭제
   - 날짜 선택 필수 (달력 연동)
   - totalPoints 자동 계산 (트랜잭션)

3. 달력 3종
   - 사용자 포인트 달력 (녹색 점)
   - 관리자 포인트 관리 (빨간 점)
   - 경기 관리 달력 (파란 점)

4. 랭킹 시스템
   - 필터: 년도 / 시즌 / 성별(전체,남자,여자) / 상위9개
   - 실시간 반영 (Stream)
   - 1~3등 색상 구분 (금/은/동)

5. 홈 스크린
   - 공지 슬라이더
   - 다음 경기 3개
   - TOP 3 랭킹
   - 뉴스 캐러셀
   - 스폰서 배너

6. 관리자 기능
   - 경기 CRUD (달력 + 수정/삭제)
   - 포인트 CRUD (달력 + 수정/삭제)
   - 사용자 검색 + 신규 등록

3. 기술 스택
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
